# ABcontroll

Android приложение для контроля Telegram-бота. Пользователь вводит API token, видит входящие сообщения и может отвечать на них прямо из приложения. Интерфейс стилизован под Telegram.

## Возможности
- Ввод токена бота
- Отображение новых сообщений
- Ответ на сообщение

## Запуск
Откройте проект в Android Studio и запустите модуль `app`.

## Примечания
Приложение использует опрос `getUpdates` Telegram Bot API.
